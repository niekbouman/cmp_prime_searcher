# cmp_prime_searcher
Search for prime numbers that enable small-range secure comparisons

# requirements
- FLINT library for number theory
- GMP (header files)

# installation
Please clone project with --recursive option to also clone the submodules

# running

    ./searcher <search offset> <k> <D0> <Dk>
